# Devhub.io SSR

> Using Next.js

## Build Setup

``` bash
# install dependencies
$ npm install

# env config
$ cp .env.example .env

# serve with hot reload at localhost:7002
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
